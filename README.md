# Student Management System (Java)

## Project Overview
This project is a simple console-based Student Management System developed using Java.
It allows users to add, view, search, and manage student records using a menu-driven interface.
The project is created as part of Week 1 task: Java Basics & Programming Fundamentals.

---

## Features
- Add new student details
- View all students
- Search student by ID
- Update student information
- Input validation for age and grade
- Menu-driven console interface

---

## Technologies Used
- Java (JDK 8 or above)
- Scanner class for user input
- Object-Oriented Programming concepts
